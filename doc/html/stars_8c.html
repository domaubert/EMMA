Define the state of a particle in function of his age
State are defined as follow:
state = 0 -> Dark Matter
state = 1 -> Radiative Star
state = 2 -> Supernovae
state = 3 -> Supernovae + decreasing luminosity
state = 4 -> Decreasing luminosity
state = 5 -> Dead star
